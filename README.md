# AI Fitness Assistant (Streamlit App)

This is a Streamlit-based AI Fitness Assistant powered by a quantized Mistral-7B model.
It includes a user profile tab, an AI chat interface, workout logging, and a clean dark UI.

Features:
User Profile (age, weight, goals, experience, injuries, etc.)

AI Chat (uses your profile + workout history as context)

Workout Log (exercise name, sets, reps, weight, date)

Achievements section (placeholder for future features)
